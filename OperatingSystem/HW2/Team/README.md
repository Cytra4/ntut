# OS Homework - Team Programming Projects

This folder contains solutions for the team-based programming project:

1. **Chapter 4 Project 2**: `/CH4_PROJ2/sorting`
2. **Chapter 5 Project**: `/CH5_PROJ/scheduler`

## Files

- `sorting.c` -> builds `sorting`

- `scheduler.c` -> builds `scheduler`

## Build

sorting.c:

```bash
gcc sorting.c -o sorting -lpthread
```

scheduler.c:
```bash
gcc scheduler.c -o scheduler -lpthread
```

- Or you can just run the executable files in each project folder

## Execution

- Chapter 4 Project 2
```bash
./sorting
```

- Chapter 5 Project 
```bash
./scheduler
```
